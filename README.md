# Woo Book Importer

![Plugin Version](https://img.shields.io/badge/version-1.7.3__gemini-blue.svg) 
![WooCommerce](https://img.shields.io/badge/WooCommerce-7.0%2B-green.svg)
![WordPress](https://img.shields.io/badge/WordPress-6.0%2B-blue.svg)

Import books directly into WooCommerce from Google Books and Amazon with complete metadata, cover images, and pricing data.

## Features

- 📚 **Single or Bulk Import** - Add books by ISBN (one or many at once).
- 🌄 **High-Res Covers** - Automatic image imports from multiple sources (Google Books & Open Library).
- 💰 **Smart Pricing** - Amazon price integration via Keepa API.
- 📊 **Real-Time Stats** - Track daily/weekly imports from the dashboard.
- 🏷️ **Auto-Categorization** - Creates and assigns product categories from book genres automatically.

## Installation

1. Upload the `woo-book-importer` folder to your `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in your WordPress admin area.
3. Navigate to **Book Importer → Settings** and enter your API keys for Google Books and Keepa.

## Requirements

- WordPress 6.0+
- WooCommerce 7.0+
- PHP 8.0+
