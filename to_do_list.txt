# Woo Book Importer

Import books directly from Google Books and Amazon into WooCommerce.

## Features

- Single or bulk ISBN imports
- Automatic high-resolution cover images
- Amazon price data via Keepa API
- Real-time import statistics
- Automatic category generation

## Installation

1. Upload the `woo-book-importer` folder to `/wp-content/plugins/`
2. Activate the plugin in WordPress admin
3. Set API keys in Settings → Book Importer
